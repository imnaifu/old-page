# canvas 知识点
可以把 canvas 当作我们自己画图的方式使用，拿起笔 `beginPath()`，找到起始点 `moveTo(x, y)`，
开始画直线 `lineTo(x, y)`，让这个线能明显被看到 `stroke()`，填充 `fill()`  
  
canvas 画图实际上只有三种
- 矩形 rect
- 弧形/圆形 arc
- 曲线（二次，三次贝塞尔曲线）
