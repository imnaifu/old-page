# Basic Command
## Window
- 新窗口：Ctrl + Shift + N
- 新文件：Ctrl + N
- 隐藏侧边栏：Ctrl + B

## Code
- 缩进：Ctrl + []
- 折叠：Ctrl + Shift + []
- 上下移：Alt + Up， Alt + Down
- 下方插入：Ctrl + Enter
- 选中当前行：Ctrl + i
- 选中匹配下一个： Ctrl + d
- 选中所有匹配：Ctrl + Shift + L
- 查找: Ctrl + f
- 替换：Ctrl + h
- 查找整个文件夹：Ctrl + Shift + f

## Key Shortcut
- 快捷键列表：Ctrl + k + s
- 主命令台：Ctrl + Shift + p
- 当前命令台：Ctrl + p
    - : go to line
    - # go to symbol by workspace 
    - @ go to symbol by file
    - ? list option
